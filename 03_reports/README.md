# 03_reports — 報告・上申

## 1. そのフォルダの目的

検証可能な根拠と未確定部分を明示した報告書・上申書を作成し、ハイエク軍およびユーザーの判断材料として引き渡す。

## 2. 保存する資料の種類

- 調査報告書、比較報告書、論点メモ、上申書の草案と確定版
- 納税者負担の集計、根拠資料一覧、計算過程への参照
- 事実、引用、出典、推定、評価、未確認事項、情報公開請求候補を分離した成果物
- レビュー記録、版情報、更新履歴、差戻し事項

## 3. Codexが行ってよい作業

- `04_templates`に基づく草案、要約、表、出典一覧の作成
- `01_sources`と`02_processed`に基づく事実関係、納税者負担、論点の整理
- 推定・評価・未確認事項を明示した比較案や反対方向の解釈の提示
- 誤記、参照切れ、集計不整合、根拠不足の検査と修正案の提示

## 4. Codexが行ってはいけない作業

- 報告書をハイエク軍またはユーザーの最終判断として確定・公表すること
- 違法性、妥当性、政治的評価、制度設計、処分、採否を最終判断すること
- 出典のない主張、架空の引用、断定的な推定を記載すること
- 不確実性、反証、未確認事項を隠して結論を誘導すること

## 5. ハイエク軍への接続方法

`04_templates/ハイエク軍_上申様式.md`等を使用し、案件ID、版、作成日、主要出典、納税者負担、未確認事項、情報公開請求候補を記載する。`00_index`から最新版へリンクし、ハイエク軍およびユーザーがレビュー、差戻し、承認、最終判断を行う。

## 6. 運用上の注意

- 「確定版」は文章の版を示すだけで、内容の法的・政治的な確定を意味しない。
- 評価は評価基準と根拠事実を示し、事実や引用と明確に分離する。
- 推定には推定表示、根拠、方法、仮定、不確実性を必ず付す。
- 補助金、助成、給付、減免、委託、指定管理、外郭団体支援は、すべて納税者負担として整理し、合計時の重複と範囲を注記する。
