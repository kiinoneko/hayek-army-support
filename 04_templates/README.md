# 04_templates — 標準様式

## 1. そのフォルダの目的

台帳、分析、報告、上申の項目と情報区分を標準化し、人間とCodexが同じ運用規則で成果物を作成できるようにする。

## 2. 保存する資料の種類

- ハイエク軍向け上申様式
- 公共選択、財政、官僚制、利益団体、選挙、多中心統治等の分析様式
- 案件台帳、出典台帳、納税者負担台帳、レビュー用チェックリストのひな型
- 事実、引用、出典、推定、評価、未確認事項、情報公開請求候補を分離する標準見出し

## 3. Codexが行ってよい作業

- 既存方針を維持した様式の複製、項目追加、表記統一、リンク修正
- 納税者負担欄、出典欄、推定条件欄、最終判断留保の追記
- 案件に適したテンプレート候補の提示と、草案への適用
- 変更案と影響範囲の提示、旧様式との比較

## 4. Codexが行ってはいけない作業

- 事実と推定、評価等を混同させる様式へ変更すること
- Codexに最終判断、承認、公表権限があるような項目を設けること
- 違法性、妥当性、政治的評価、制度設計の結論をあらかじめ埋め込むこと
- 既存成果物との互換性を確認せず、必須項目を削除または改名すること

## 5. ハイエク軍への接続方法

`00_index`、`02_processed`、`03_reports`で本フォルダの様式を使用する。上申時は原則として`ハイエク軍_上申様式.md`を基礎にし、必要な分析様式を添付資料として接続する。様式の変更は、適用日または版を明示して共有する。

## 6. 運用上の注意

- 様式は思考補助であり、空欄を推測で埋めるためのものではない。
- すべての様式で、補助金、助成、給付、減免、委託、指定管理、外郭団体支援、公費投入、施設維持費、行政サービス拡充を納税者負担として扱う。
- 推定欄には根拠、方法、仮定、不確実性を要求し、断定表現を避ける。
- 最終判断がハイエク軍およびユーザーに留保されることを維持する。

## 7. 実戦投入用様式

- [Codex軍令 様式](Codex軍令_様式.md)：入力資料、成果物、保存先、情報加工段階、禁止事項およびPR確認観点を指定し、後方支援軍からCodexへ作業命令を出す。
- [ハイエク軍 フィードバック対応様式](ハイエク軍_フィードバック対応様式.md)：ハイエク軍または各奉行から不足指摘を受けた際の再召集、追加成果物、PR確認および再上申を管理する。

両様式は、Codexおよび後方支援軍に最終判断を委ねるものではない。原資料、出典、加工経路、確度を維持し、ハイエク軍または各奉行が評定できる状態へ整えるために使用する。
