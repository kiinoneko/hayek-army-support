## 10. 納税者負担の扱い

補助金、助成、給付、減免、委託、指定管理、外郭団体支援、公費投入、施設維持費、行政サービス拡充は、名称、会計区分または「無料」「無償」「支援」等の表現にかかわらず、**すべて納税者負担として整理する**。

- 金額、年度、財源、支出主体、受益者、負担者、支出根拠を可能な範囲で分ける。
- 金額等が確認できない場合は、空欄の意味が不明にならないよう「未確認」と記載する。
- 国・自治体間移転、再委託、複数制度等による重複計上の可能性を点検する。
- 納税者負担に該当しないと判断した項目も、除外根拠を記録する。

## 11. 事実・推定・評価の分離

成果物では、以下を見出し、列、注記または別ファイルによって**必ず分離する**。

| 区分 | 記載要件 |
|---|---|
| 事実 | 原資料で確認できる内容のみを記載する。 |
| 引用 | 原文と要約を区別し、引用箇所を特定できるようにする。 |
| 出典 | 資料名、公開元、出典URL、公開日、取得日、保存場所等を示す。 |
| 推定 | 推定であることを明記し、根拠、方法、仮定、不確実性を添える。 |
| 評価 | 誰の評価かを示し、確認済み事実と混同しない。Codex自身の最終評価は記載しない。 |
| 未確認事項 | 未確認の内容、確認に必要な資料、確認方法を示す。 |
| 情報公開請求候補 | 請求対象、保有機関、必要な文書または項目、対象期間を可能な範囲で示す。 |

## 12. 禁止事項

Codexは、次の事項を行ってはならない。

- [ ] 最終判断
- [ ] 違法性の断定
- [ ] 不正の断定
- [ ] 政治的意図の断定
- [ ] 出典不明情報の事実化
- [ ] 推定の事実化
- [ ] ハイエク軍最終評定の代行
- [ ] 各奉行の結論の先取り
- [ ] グラフや資料による印象操作
- [ ] 原資料へ戻れない加工

疑義がある場合は断定せず、確認済み事実、疑義が生じる理由、代替説明、未確認事項および追加確認方法として整理する。

## 13. PR確認観点

PR確認では、少なくとも以下を確認する。

- [ ] 指定ファイルが作成されているか
- [ ] 保存先が正しいか
- [ ] 出典があるか
- [ ] 原資料へ戻れるか
- [ ] 納税者負担が整理されているか
- [ ] 事実・推定・評価が分離されているか
- [ ] Codexが最終判断まで踏み込んでいないか

### 追加確認事項

- [ ] ファイル名、列、見出し、文字コード等が軍令の指定どおりである。
- [ ] 原資料と加工物の対応および加工経路を追跡できる。
- [ ] 未確認事項と情報公開請求候補が削除されず、明示されている。
- [ ] 図表には出典、単位、対象期間、注記がある。
- [ ] 重複計上、期間差、会計区分差、単位差が点検されている。
