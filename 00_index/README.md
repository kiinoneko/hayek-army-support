# ハイエク軍 後方支援軍インデックス

このリポジトリは、ハイエク軍へ提出する入力資料を収集・整理するための作業領域です。

## ディレクトリ

- `00_index`: 案件一覧、索引、進捗管理
- `01_sources`: 原資料、公開資料、取得記録
- `02_processed`: 原資料から抽出・整形した作業資料
- `03_reports`: テンプレートに基づく報告書
- `04_templates`: 分析・上申用Markdownテンプレート
- `05_tools`: 収集、整形、検証を補助するツール

## 運用原則

1. 事実、引用、出典、推定、評価、未確認事項、情報公開請求候補を混在させず、明示的に区分する。
2. 補助金、助成、給付、減免、委託、指定管理、外郭団体支援は、名称や会計上の区分にかかわらず、すべて納税者負担として整理する。
3. 引用には、引用箇所、資料名、作成主体、公開日または取得日、ページまたはURLを記録する。
4. 推定には根拠、計算式、仮定、不確実性を付記する。
5. Codexは資料の収集、区分、整形、論点提示までを担い、政策・政治・法的な最終判断を行わない。
6. 最終判断はハイエク軍が行い、本リポジトリの成果物はそのための入力資料として扱う。
